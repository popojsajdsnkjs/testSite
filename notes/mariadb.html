Powrót

Słownik

  • NOW() - jako timestamp daje teraźniejszą date

Syntax

  • INSERT
    INSERT INTO nazwa tabeli ( nazwa kolumny , nazwa kolumny ,...)
    VALUES ( dane do kolmny )
  • CREATE TABLE
    CREATE TABLE nazwa tabeli ( nazwa kolumny typ danych dodatkowe szczegóły takie jak NOT NULL, AUTO_INCREMENT etc. odzielone spacją )

    Argumenty do kolumn (można użyc ich w create i alter table)

    • AUTO_INCREMENT - auto numerowanie
    • NOT NULL - nie NULL
    • PRIMARY KEY - klucz podstawowy
    • FOREIGN KEY - klucz obcy, do relacji
      REFERENCES - do relacji
    • UNIQUE - dane się nie powtórzą w kolumnie
    • DEFAULT - domyślna wartość dla kolumny

ALTER TABLE - modyfikowanie tabeli

  • Dodanie nowej kolumny
    ALTER TABLE nazwa tabeli ADD COLUMN nazwa kolumny typ danych;
  • Usunięcie kolumny
    ALTER TABLE nazwa tabeli DROP COLUMN nazwa kolumny;
  • Zmiana typu danych kolumny
    ALTER TABLE nazwa tabeli MODIFY COLUMN nazwa kolumny nowy typ;

SELECT

  • Filtrowanie danych WHERE
    SELECT * FROM nazwa_tabeli WHERE kolumna = 'wartość';
  • Sortowanie ORDER BY
    SELECT * FROM nazwa_tabeli ORDER BY kolumna ASC/DESC;
  • Ograniczenie liczby wyników LIMIT
    SELECT * FROM nazwa_tabeli LIMIT 2;

Tworzenie kopii zapasowej DUMP

Poza mysql, w czystym cmd (folder c:/xampp/mysql/bin)

  • Eksport bazy danych do pliku
    mysqldump -u użytkownik -p nazwa_bazy > backup.sql
  • Import bazy danych z pliku
    mysql -u użytkownik -p nazwa_bazy < backup.sql
